# 职业路径

## 产品经理的典型成长路径

```
产品助理/专员
    ↓ (1-2年)
产品经理
    ↓ (2-3年)
高级产品经理
    ↓ (2-3年)
产品专家 / 产品总监
    ↓ (3-5年)
产品副总裁 / CPO
```

## 各阶段能力要求

### P0 - 产品助理 (0-1年)
**目标**：熟悉流程，做好执行

| 能力 | 要求 |
|------|------|
| 文档能力 | 基础的 PRD 撰写 |
| 沟通能力 | 准确传达需求 |
| 工具熟练度 | Figma/Axure、流程图 |
| 执行力 | 按要求完成任务 |

### P1 - 产品经理 (1-3年)
**目标**：独立负责模块

| 能力 | 要求 |
|------|------|
| 需求分析 | 能挖用户真实需求 |
| 数据分析 | SQL、漏斗、A/B 测试 |
| 项目管理 | 推动功能按时上线 |
| 方法论 | 形成自己的分析框架 |

### P2 - 高级产品经理 (3-5年)
**目标**：负责整条产品线

| 能力 | 要求 |
|------|------|
| 产品规划 | 制定 Roadmap |
| 商业思维 | 理解商业模式 |
| 领导力 | 带新人、跨团队推动 |
| 行业洞察 | 对市场有深刻理解 |

### P3 - 产品总监/专家 (5-8年)
**目标**：制定产品战略

| 能力 | 要求 |
|------|------|
| 战略思维 | 从公司战略拆解产品方向 |
| 团队管理 | 搭建团队、培养人才 |
| 资源调配 | 在有限资源下最大化产出 |
| 影响力 | 在产品方向上有决策权 |

### P4 - CPO / VP Product (8年+)
**目标**：业务操盘，定义方向

## 如何加速成长

### 日常积累
- **阅读**：行业报告、产品博客、书籍
- **拆解**：新上线功能第一时间体验和分析
- **复盘**：每个项目结束后写复盘文档

### 建立影响力
- 写产品分析文章/博客
- 参加行业交流
- 内部知识分享

## 跳槽与薪资

| 层级 | 薪资范围（一线城市） | 跳槽频率 |
|------|---------------------|----------|
| 助理/专员 | 8-15K | 2-3年 |
| 产品经理 | 15-25K | 2-3年 |
| 高级 PM | 25-40K | 3-4年 |
| 产品总监 | 40-70K+ | 4-5年 |

> 以上为参考范围，视行业、公司规模和面试表现差异较大。
